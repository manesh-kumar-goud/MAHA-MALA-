## üéØ Design Principles

### Spacing & Layout
- **Generous padding**: 32-48px (py-32, py-24)
- **Max width**: 7xl (1280px) for content
- **Section spacing**: Large gaps between sections
- **Card padding**: 24-32px (p-6, p-8)

### Animations
- **Subtle entrance**: Fade in + slide up
- **Hover effects**: Slight lift, border color change
- **Smooth transitions**: 300ms duration
- **Scroll animations**: Elements animate on scroll

### Components Style
- **Buttons**: 
  - Primary: Teal background, white text, rounded
  - Secondary: Outlined, teal border
  - Hover: Slight shadow, color change
- **Cards**: 
  - White background, subtle border (teal-200)
  - Rounded corners
  - Hover: Border color change, slight lift
- **Icons**: 
  - Teal color (#0D9488)
  - Consistent size (h-6 w-6 or h-7 w-7)
  - In circular backgrounds (teal-100)

### Responsive Design
- **Mobile-first**: Stack vertically on mobile
- **Breakpoints**: sm (640px), md (768px), lg (1024px), xl (1280px)
- **Touch-friendly**: Minimum 44x44px touch targets
- **Horizontal scroll**: For card sections on mobile

### Visual Elements
- **Icons**: Lucide React icons style (line icons)
- **Images**: High-quality, professional solar installations
- **Patterns**: Subtle dot patterns for backgrounds (very low opacity)
- **Gradients**: Subtle teal gradients for backgrounds
- **Shadows**: Minimal, soft shadows
