## üìù Notes

- All images will show a placeholder if the file doesn't exist
- Images should be optimized for web (compressed, appropriate file size)
- Recommended format: JPG for photos, PNG for graphics with transparency
- All images are responsive and will scale automatically
- Testimonial images will automatically fallback to initials if image is missing

## üé® Image Optimization Tips

1. **Compress images** before uploading (use tools like TinyPNG, ImageOptim)
2. **Use appropriate dimensions** - don't upload unnecessarily large images
3. **Maintain aspect ratios** - especially for testimonial photos (square)
4. **Use descriptive filenames** - helps with SEO and organization
5. **Consider WebP format** - better compression, but ensure fallback support
